=== GitHub Pages 有効化手順 ===

1. GitHubリポジトリにプッシュ
   git add docs/
   git commit -m "Add client setup guide webpage"
   git push origin main

2. GitHubリポジトリの Settings を開く

3. 左メニューから「Pages」を選択

4. Source で「Deploy from a branch」を選択

5. Branch で「main」を選択、フォルダで「/docs」を選択

6. Save をクリック

7. 数分後、以下のURLでアクセス可能:
   https://[あなたのGitHubユーザー名].github.io/Minecraft_HcSl-Server/

8. Discord投稿用メッセージの [URL] 部分を実際のURLに置き換えて投稿
